    Facial Analysis - Emotion Recognition

    Technologies Used: Tensorflow, Python, AngularJS, EC2-AWS

    This project lays the foundation for a bigger project that aims at "Capturing People Emotions in Realtime while they are watching online videos".

    A convolutional neural network is trained to ouput a certain confience across a set of emotions for a given input image.

    Smart Office

    Technologies Used: Raspberry Pi, Arduino, Zigbee, Apache, PHP, JS

    In order to reduce wastage of electricity at the student activity centre, built a wireless network of sensors and actuators to monitor and control all the appliances over a web application. The sensor data retrieved in real time was fed into a learning algorithm to understand the usage pattern and control the devices autonomously. The energy consumption decreased by 12%.

    Easy Charge Management for Laptops

    Technologies Used: Shell, Arduino, Arduino-Wifi

    An application to maintain battery charge of laptops in an optimal range between 30% and 90%

  • Users (Linux OS) had to install a shell application.
  • The shell script constantly monitored the laptop's battery percentage.
  • On the event when the charge went below 30% or above 90%, the script signalled a smart switch to turn 'on' or turn 'off' respectively.
  • The smart switch was a relay module controlled using an Arduino Mega and Arduino Wifi Module.
